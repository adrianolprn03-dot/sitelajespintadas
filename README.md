# Portal Institucional - Lajes Pintadas

Este é o portal oficial da Prefeitura Municipal de Lajes Pintadas - RN.

- [Next.js](https://nextjs.org)
- [Prisma](https://prisma.io)

- [Vercel](https://vercel.com)
